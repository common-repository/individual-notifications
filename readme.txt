== Description ==
Simple plugin to show notifications, warnings and alerts inside your posts.

== Current Options ==
* Type [in_notify]Did you know that you can bookmark this site?[/in_notify] for a notification
* Type [in_alert]Don't forget to bookmark this site![/in_alert] for an alert
* Type [in_warn]If you don't boomark this site, I shall hunt you down..[/in_warn] for a warning
